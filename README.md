# 微信群聊总结助手 (WeChat Message Summary)

一个基于人工智能的微信群聊消息总结工具。通过对接本地 WeFlow HTTP API 获取聊天记录，并利用大语言模型（如 DeepSeek、Kimi、通义千问等）生成结构化的每日/实时群聊总结。

## 功能特点

- **多模型支持**：支持所有兼容 OpenAI 接口协议的 AI 服务（如 DeepSeek, OpenAI, 阿里通义, 百度文心等）。
- **智能总结**：自动提取群聊重点内容，生成包含核心话题、重要决定、待办事项的 Markdown 结构化报告。
- **灵活触发**：
  - **手动总结**：在 GUI 界面自定义时间范围（小时/分钟）和条数限制，即时生成（默认仅预览，不自动落盘）。
  - **定时任务**：每日固定时间自动拉取过去 24 小时消息并生成总结。
- **自动化管理**：
  - **本地归档**：仅定时任务会按群名分类自动保存为 Markdown 文件。
  - **Git 同步**：支持定时任务完成后自动将总结推送到 GitHub 仓库，方便多端同步查看。
- **微信推送**：
  - **手动推送**：生成总结后可点击“推送至微信”按钮主动发送。
  - **定时自动推送**：在“定时任务配置”中启用后，定时任务总结完成自动推送到微信。
- **高度可定制**：内置提示词（Prompt）管理器，支持针对不同群聊保存不同的总结风格。
- **现代化 UI**：基于 PySide6 构建的简洁、专业的 Windows 原生界面。

## 安装说明

### 环境要求
- **操作系统**: Windows 10/11
- **Python**: 3.10+
- **微信版本**: 微信 4.0 以下（需配合特定的 WeFlow/wxauto 环境）
- **WeFlow**: 需安装 [WeFlow](https://github.com/hicccc77/WeFlow) 并开启 HTTP API 服务（默认端口 5031）。

### 依赖安装

建议使用 `uv` 或 `pip` 安装依赖：

```bash
pip install -r requirements.txt
```

### 快速开始

1. **配置环境变量**：在根目录创建 `.env` 文件（或修改现有文件）：
   ```env
   BASE_API_URL=http://127.0.0.1:5031/api/v1
   ```
2. **启动程序**：
   ```bash
    python main_gui.py
   ```

## 使用指南

### 1. AI 服务配置
- 切换到 **"AI 服务配置"** 标签页。
- 添加您的 API Key 和 Base URL。
- 可管理多个服务（如工作用 GPT-4，个人用 DeepSeek）。

### 2. 提示词管理
- 切换到 **"提示词配置"** 标签页。
- 您可以根据需要修改 AI 的总结风格（例如：更详细、更简练、或专注于技术讨论）。

### 3. 生成总结
- 在 **"群聊总结"** 界面输入群聊名称关键字。
- 设置获取范围（如：最近 2 小时 30 分钟）。
- 点击 **"生成总结"**。程序会自动分页拉取所有消息，确保无遗漏。
- 生成成功后：
  - 可点击 **"导出总结"** 手动导出 Markdown。
  - 可点击 **"推送至微信"** 将当前预览总结完整推送到微信（需先在“定时任务配置”中填写 Webhook）。

### 4. 定时任务、Git 与微信推送
- 在 **"定时任务配置"** 开启每日自动总结。
- 如需自动同步到 GitHub，配置 Access Token 和仓库地址后会执行 `git pull --rebase` -> `commit` -> `push`。
- 如需自动推送到微信，在同一页面配置：
  - 启用“定时总结后自动推送到微信”
  - `Webhook 地址`
  - `Webhook 密钥`
  - 微信推送Webhook如何获取 请查看 [wxWebHook](https://github.com/aristorechina/wxWebHook)

## 目录结构

- `main_gui.py`：图形界面入口。
- `wechat_summary.py`：核心业务逻辑（API 调用、AI 处理）。
- `config/`：存储 AI、提示词、常规设置及定时任务配置 (JSON)。
- `summary/`：定时任务生成的总结文件存放地。
- `logs/`：程序运行日志。
- `icons`：程序图标资源。
- `.env`：本地 API 地址配置文件。
- `requirements.txt`：项目依赖清单。
